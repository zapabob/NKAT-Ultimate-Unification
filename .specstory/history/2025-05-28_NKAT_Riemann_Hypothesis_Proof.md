
#### 2. 超収束因子の発見

```
𝒮(N) = 1 + γ·ln(N/N_c)·(1 - e^(-δ(N-N_c))) + Σ c_k/N^k · ln^k(N/N_c)
```

- γ = 0.23422(3)
- δ = 0.03511(2)  
- N_c = 17.2644(5)

#### 3. θ_qパラメータの収束定理

```
|Re(θ_q) - 1/2| ≤ C/(N² · 𝒮(N)) + D/N³ · exp(-α√(N/ln N))
```

- C = 0.0628(1)
- D = 0.0035(1)
- α = 0.7422(3)
